# LSMProject

Платформа для проведения курсов в рамках проекта «Код будущего».

## Что уже подготовлено

- первичный анализ требований федерального LMS-проекта;
- стартовый план разработки;
- структура контекста для Claude с упором на низкий расход токенов;
- базовые документы по требованиям, архитектуре и тестированию.

## Структура

- `CLAUDE.md` — краткие правила работы агента в репозитории;
- `.claude/context/` — короткие контекстные файлы, которые можно читать точечно;
- `docs/` — продуктовая и инженерная документация.
- `apps/api/` — backend API на FastAPI;
- `apps/web/` — web-интерфейс на Next.js;
- `infra/docker/` — локальный docker compose для старта сервисов.

## Быстрый старт

### Backend
- создать и активировать виртуальное окружение;
- установить зависимости из `apps/api`;
- запустить `uvicorn app.main:app --reload` из каталога `apps/api/src` или через настроенную команду окружения.

### Frontend
- установить Node.js 22+ и `npm`;
- установить зависимости: `npm --prefix apps/web install`;
- запустить: `npm --prefix apps/web run dev`.

### Проверка
- backend healthcheck: `GET /api/v1/health`;
- backend overview: `GET /api/v1/overview`;
- web: стартовая страница платформы на порту 3000.

## Что реализовано на старте

- monorepo-структура проекта;
- FastAPI backend scaffold;
- Next.js frontend scaffold;
- базовый CI workflow;
- docker compose для локального запуска;
- healthcheck и обзорный API endpoint.
